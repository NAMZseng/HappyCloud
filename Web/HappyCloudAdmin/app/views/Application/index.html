欢喜云盘后台管理员界面，实现的功能如下：
------------------------------------------------------------------------------------
(1) 登录权限管理，管理员输入密码才可进行相关操作；
(2) 表操作，可对系统相关数据库表进行增删查改操作；
(3) 多种方式查询，对于user_info表，可通过user_id，name，phone进行用户查询，
　 对于user_download/share_file表，可通过file_id，user_id进行记录查询；
(4) 提供Hadoop分布式文件系统(HDFS)控制页面的访问跳转；

HappyCloud Administor Page built with Play framework By Nanrong_Zeng.
